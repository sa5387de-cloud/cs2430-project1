# cs2430-project1

## How to Run
1. Download `main.py` (or download the repo as ZIP).
2. Open Command Prompt (cmd).
3. Go to the folder containing `main.py`:
   - If on OneDrive Desktop:
     `cd %USERPROFILE%\OneDrive\Desktop`
4. Run:
   - `py main.py`

## What This Program Does
### Part 1: Ordinary Sets (Boolean Arrays)
Computes and prints:
- NOT(A)
- A ∪ B
- A ∩ B
- A − B
- A ⊕ B (symmetric difference)

Outputs include both:
- Boolean array representation
- Element listing

### Part 2: Multisets (Bags)
Computes and prints:
- Multiset union (max counts)
- Multiset intersection (min counts)
- Multiset difference (A − B, not below 0)
- Multiset sum (A + B)
